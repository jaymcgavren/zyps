Zyps

About Zyps

Zyps is a Ruby library for creating games with creatures that act on their own.

Getting started is easy; just code the way you want objects to react to each other. The Zyps library will handle the drawing, timing, interaction, and everything else.

Screenshots

You can add environmental factors like wind or gravity.

Download

You'll need Ruby (http://www.ruby-lang.org) and the Ruby-GNOME2 library (http://ruby-gnome2.sourceforge.jp).

Once those are installed, just type this at a command line:

gem install zyps
